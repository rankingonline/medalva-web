# Medalva Web Project

Este proyecto es el punto de partida (scaffolding) con la estructura solicitada.

## Estructura de Directorios
- `assets/img/` - Contiene las imágenes y logos (ej. `logo.svg`).
- `src/about_us/` - Contiene los archivos HTML, CSS, y JS para la sección "About Us".
- `index.html` - El punto de entrada principal que redirige o enlaza a la página de "About Us".
